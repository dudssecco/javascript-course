npm init -y:                        Iniciar o node
npm install (nome):                 instalar biblioteca / pacote
npm install express:                instalar express
npm install express --save-dev:     instalar express nas DevDependences
npm install express --save-prod:    instalar express nas Dependences
npm install express@2.1.0:          instalar o express na versão 2.1.0
npm install express@2.1.0 -E:       instalar versão especifica sem atualização
npm unistall express:               desistalar express
